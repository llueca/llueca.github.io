Several years ago I had a plan. To demonstrate my skills as web developer, I should create my own webpage from the scratch. No Wordpress, no Drupal… just plain (now so) old PHP.

Because, if I pretend to be a good web developer, I should be able to show it without sharing any code of projects where I’ve been. Something as ilegal as unprofessional.

Ok. Today I’ve decided that maybe it should exists another way to show my skills with something less time consuming. Maybe I can recover my original idea somewhen in the future, but in the meanvwhile I would be able to show to the world what I can do.

Today I create this blog to share my findings, my thoughts, my code… all about this profession that I really love. Gatsby will help me by now.

I hope you enjoy your stay in this site.
